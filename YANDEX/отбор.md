Строим пауком карту сайта.
Замечаем странную передачу параметра style, параметр можно поменять, надо проверить на LFI

![](../images/Pasted%20image%2020260311114138.png)

Подставляем разные полезные нагрузки LFI и находим стандартную работающую:

```
../../../../etc/passwd
```

Видим, что среди другого html кода отображается содержимое passwd

![](../images/Pasted%20image%2020260311114321.png)

Просматриваем запущенные приложения, чтобы узнать имя файла запуска сервера
```
/proc/self/cmdline
```
В ответе видим название файла нашего сервера

![](../images/Pasted%20image%2020260311114706.png)

Начиная с ../../../../app.py и постепенно уменьшая количество ../ ищем исходный файл
Видим, что он находится в pwd и таким образом мы получили исходный код python веб-сервера

![](../images/Pasted%20image%2020260311114906.png)

Из исходного кода узнаем, что токен JWT подписывается ключом r@nd0m_s3ctr3t_v4luE_GoGogo&#34
И чтобы получить флаг нам надо зайти в /developer
Создаем свой куки JWT токен на сайте

![](../images/Pasted%20image%2020260311115557.png)

Делаем запрос на ip/developer подставив новый куки и находим флаг

![](../images/Pasted%20image%2020260311115709.png)
